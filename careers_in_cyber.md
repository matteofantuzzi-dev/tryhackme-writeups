# Careers in Cyber Security

## Ruoli principali
- SOC Analyst
- Blue Team Engineer
- Red Team / Penetration Tester
- Security Engineer
- Incident Responder

## Cosa fa ogni ruolo (in breve)
- SOC Analyst: monitors alerts, analyzes logs, detects incidents
- Blue Team: focuses on defense, monitoring, and hardening systems
- Red Team: simulates attacks to find vulnerabilities
- Security Engineer: designs and maintains secure infrastructures
- Incident Responder: handles security incidents and recovery

## Competenze comuni richieste
- Networking fundamentals
- Linux and Windows basics
- Understanding of logs and monitoring
- Security mindset
