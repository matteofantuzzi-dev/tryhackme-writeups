# Linux Fundamentals

## Part 1 – Basic Commands
- pwd: print current directory
- ls: list files and folders
- cd: change directory
- mkdir / rmdir: create / remove directories
- cp / mv / rm: copy, move, remove files
- cat / less / more: view file contents

## Part 2 – File Permissions
- chmod: change file permissions
- chown: change file owner
- rwx: read, write, execute for user/group/others
- Understanding permissions is key for system security

## Part 3 – Processes and Users
- ps / top / htop: view running processes
- kill / killall: terminate processes
- who / w / id: check logged in users
- sudo: run commands as superuser
- Understanding processes and users is essential for administration and security

## Key Takeaways
- Linux commands are the foundation for system management and security
- File permissions and user management are critical for hardening systems
- Process monitoring helps in troubleshooting and security analysis
