# Networking Fundamentals

## What is Networking
- Networking allows devices to communicate and share data
- Data is transferred using protocols
- Networks can be public or private

## Intro to LAN
- LAN stands for Local Area Network
- Devices in a LAN communicate using switches and routers
- IP addresses identify devices on the network
- MAC addresses identify network interfaces

## Key Concepts
- IP Address
- MAC Address
- Switch vs Router
- Packet-based communication

## Why this matters in Cyber Security
Understanding networking is essential to analyze traffic,
identify attacks, and secure systems.
